## Additional information
Here's a small selection of my Processing projects, and I plan to continue adding to it over time. I've cleaned up the code for each project, stripping away any code from my template that wasn't being used. I've added some comments to parts of the code that I thought might need a bit of explanation.

I started making these projects in late 2018. In the beginning, I drew inspiration from [Dave @beesandbombs](https://beesandbombs.com/) and [Etienne @bleuje](https://bleuje.com/), learning techniques and approaches from them. Although this wasn't my first time coding, it was through these projects that I really began to develop my coding skills and understanding. Over the years my skills have significantly improved with these projects, as they've repeatedly challenged and tested my problem-solving abilities in various ways.

You can find a gallery with most of my cool processing animations here on my website: [https://jn3008.com/processing/](https://jn3008.com/processing/)
